## ancast-sigcheck  
Verifies the validity of a given ancast image.  
Not all the signature are supported yet, and some checks might be missing.  
  
### Usage  

    ancast-sigcheck <ancast_image>
  
### Building  
Requires openssl. Run `make` to build the program.
